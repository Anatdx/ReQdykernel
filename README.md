# 这是个人制作的修改版秋刀鱼内核
## 鉴于原版秋刀鱼内核使用了非开源程序构建，以及对内核镜像进行所谓“防倒卖”的加密，
## 要求用户在设备上以root权限执行可疑程序，将可疑程序打包进AK3包等，
## 作者认为这种行为并不友好，因此制作了这个修改版，将可疑程序与加密部分全部替换。

## 仅在OnePlus Ace 5 Ultra上测试过，其他设备不保证正常使用。

### 由于作者精神状态不佳，可能无法及时更新，敬请谅解。
